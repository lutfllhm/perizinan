# Railway MySQL Setup Guide

## Langkah-langkah Setup MySQL di Railway

### 1. Pastikan MySQL Plugin Sudah Ter-install

Di Railway Dashboard:
1. Buka project Anda
2. Klik **"+ New"** 
3. Pilih **"Database"** → **"Add MySQL"**
4. MySQL service akan dibuat otomatis

### 2. Link MySQL ke Backend Service

1. Buka **backend service** di Railway
2. Pergi ke tab **"Settings"**
3. Scroll ke bagian **"Service Variables"** atau **"Connected Services"**
4. Pastikan MySQL service sudah ter-link
5. Jika belum, klik **"Connect"** dan pilih MySQL service

### 3. Verifikasi Environment Variables

Setelah MySQL ter-link, variables berikut akan otomatis tersedia:

```
MYSQLHOST=mysql.railway.internal (atau hostname lain)
MYSQLPORT=3306
MYSQLUSER=root
MYSQLPASSWORD=<auto-generated>
MYSQLDATABASE=railway
```

**Cara Cek:**
1. Buka backend service
2. Pergi ke tab **"Variables"**
3. Pastikan semua variable MySQL di atas ada
4. Jika tidak ada, berarti MySQL belum ter-link dengan benar

### 4. Tambahkan Variables Tambahan

Di tab **Variables**, tambahkan:

```
NODE_ENV=production
PORT=5000
JWT_SECRET=<your-jwt-secret>
FRONTEND_URL=<your-frontend-url>
```

### 5. Redeploy Backend

Setelah semua variables ter-set:
1. Klik **"Deploy"** atau push ke GitHub
2. Railway akan otomatis redeploy
3. Check logs untuk memastikan koneksi MySQL berhasil

## Troubleshooting

### Error: ENOTFOUND mysql.railway.internal

**Penyebab:** MySQL service belum ter-link atau variables tidak ter-inject

**Solusi:**
1. Pastikan MySQL plugin sudah ter-install
2. Link MySQL ke backend service
3. Restart backend service
4. Check variables di tab Variables

### Error: Access denied for user

**Penyebab:** Password atau user salah

**Solusi:**
1. Check variable `MYSQLUSER` dan `MYSQLPASSWORD` di Railway
2. Pastikan tidak ada hardcoded credentials di code
3. Gunakan variables dari Railway

### Error: Table doesn't exist

**Penyebab:** Database belum diinisialisasi

**Solusi:**
Server akan otomatis membuat tables saat startup. Check logs untuk:
- ✅ Table users ready
- ✅ Table pengajuan ready
- ✅ Default admin user created

## Testing Koneksi

Untuk test koneksi MySQL dari Railway CLI:

```bash
# Login ke Railway
railway login

# Link ke project
railway link

# Run command di Railway environment
railway run node backend/scripts/init-railway-db.js
```

**CATATAN:** Command `railway run` akan menggunakan Railway environment variables, bukan local `.env` file.
